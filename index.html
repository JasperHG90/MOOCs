Welcome!

This repository contains documentation and code to store, query and pre-process Coursera MOOC data. It was created for those who have access to such data, but lack the necessary infrastructure to be able to easily query / process it. If you have any questions, comments, concerns, or if you would like to collaborate, please contact us at j.h.ginn@cdh.leidenuniv.nl.

What you can find in this repository . . .

This repository contains scripts and documentation that mainly focus on (re)storing Coursera data dumps by storing them in MySQL, SQLite and MongoDB databases. It further focuses on querying that data from the R statistical Language & Python. Finally, the repository contains a number of convenience functions to help you access and process your data such that it is ready for analysis.

The repository also includes a vagrant box, which will allow you to set up a virtual environment containing all the necessary tools to be able to run the scripts.

. . . what we will be adding later

We will be updating the repository regularly to include analyses, methods and more documentation.

Downloading the repository

Please read the readme before downloading the repository.

You can download the repository by downloading it as a .zip file or tar ball. Alternatively, you can clone the repository by installing git, opening up a terminal, and typing the following code:

# Navigate to a folder in which you want to clone the repo
$ cd /users/jasper/desktop
# Clone the repo;
$ git clone https://github.com/JasperHG90/MOOCs.git
